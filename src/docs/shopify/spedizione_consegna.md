# 🚚 配送与运费 (Spedizione e consegna) - 国际物流与分级运价指南

高效的运费配置（按重量、按价格、定区配对）是独立站运营者控制物流毛利和解决物理丢件纠纷的最核心底盘。

---

## 1. 运费规则树 (Regole di Tariffa)
在 AI Commerce OS 物流后台中，运费可以通过以下两个核心维度进行复合公式配置：
- **按订单金额 (Flat rate by Price)**:
  - 示例：订单金额 €0 - €49.99 收取固定标准运费 €4.99。
  - 示例：订单金额大于 €50 自动激活免运费方案。
- **按货品物理实重 (Flat rate by Weight)**:
  - 针对重物、美妆瓶体或贵金属，根据总 SKU 重量累计，分级匹配 DHL Premium 或 UPS Express 运价单，有效防止物流亏损。

## 2. 跨国配送区域 (Zone di spedizione)
- 欧盟内部各经济体对于货物流通实施单一市场免关税待遇。
- **自定义分级分区**:
  - **Zone Class A (Zone-EU)**: 包含德、法、意、西、荷。运达时间一般 3-5 天，运费费率较低。
  - **Zone Class B (Zone-EXTRA)**: 包含英国、瑞士、挪威等非单一关税同盟。自动追加进关申报费和运费溢价。
  - **Zone Class C (Zone-GLOBAL)**: 全球通用。

## 3. 本地化派配与门店自提 (Consegna locale e Ritiro)
- **本地配送 (Local Delivery)**:
  - 针对保税仓或中心店铺 15km 辐射地理邮编号码。可自主设置本地专项自配运费与安全送达注意事项（如：留置信箱或入户电梯密码）。
- **到店自提 (Local Pickup)**:
  - 买家可前往指定的物理 Location (Sede) 门店自提。顾客在结账时可一键选配就近网点，货物准备妥贴后由系统下发自提二维码、电子引路单至买家。

## 4. 包裹包装箱档案管理
- 在物流设置子模块中加入默认瓦楞纸包装箱的尺寸 (`Length × Width × Height`) 与纸箱附加净重。
- 系统可根据购物车中所有 SKU 的堆积体积和理论重量，智能预测适合哪款箱型装箱，提供准确的第三方动态 API 运费实价查询。
