# 📧 邮件通知 (Notifiche) - 智能业务触发与文案定制指南

交易型电子邮件 (Transactional Emails) 具有超过 80% 的行业超高开启率。精准、温和且具有即时对账价值的电邮，能最大程度提拉复购率并树立值得信赖的出海心智。

---

## 1. 全周期交易型邮件触发树
AI Commerce OS 在特定核心业务事件被成功触发并将数据写入底层数据库时，会自动启动后台邮件网关：
- **订单付款确认 (Order Payment Confirmed)**:
  - 消费者付款瞬时，系统自动拉取消费者电邮，渲染并发送涵盖所买 SKU 详情、实际 VAT 金额、收货物理地址及客服聊天直通入口的对账电邮。
- **物流出发承载 (Fulfillment Waybill Issued)**:
  - 物流仓库核销打单，获取国际 DHL 或 UPS 的实盘快递号后，系统秒级触发发货通知信，附带直达追踪连结和预计派配时效。
- **退款与调整批准 (Refund & Return Approved)**:
  - 财务在订单中心下发人工审批放款指令，自动通知信告之买家资金已回原支付卡片，缓解客服物理压力。

## 2. 废弃购物车智能挽救 (Abandoned Cart Retrieval)
- 德国及法国消费者具有高比例的在最末期结账阻滞或比价行为。
- **防止客户流失机制**:
  - 商机结账输入电邮但最终退避的 45 分钟后，AI 系统将智能编排起草一份带有专属 10% OFF 的一次性特殊面码电邮。经实测，该自动流平均可替独立站挽回 7% 至 12% 的流失销售额。

## 3. 发件人验证 (Sender Authentications)
- 为了防范电邮被收件方邮件服务供应商（如 Gmail, Outlook, Orange）归为“滥发垃圾邮件 (SPAM)”，商家必须正确设置对公发件发信人名称和邮箱（如 `support@aurora-retail.eu`）。
- **DKIM 与 SPF 验证**:
  - 推荐注册 Verified Domain。在域名 DNS 层中加入专属 `TXT` (SPF/DKIM) 代码序列，彻底提升发信箱高信誉发件到达率。
