# 🗄️ 礼品卡与元数据 (Carte regalo, Metafield e Lingue) - 特殊资产与高级定制指南

本模块包含 Shopify 极其高阶的核心设置项：礼品卡（虚拟资产溢价）、自定义模型元数据 Metafields（数据图谱扩展）以及多语言对账翻译。

---

## 1. 礼品卡管理与有效期 (Carte regalo)
虚拟电子礼品卡是在复活节、感恩节及圣诞季等欧洲重要假期提振独立站现金流和老客锁死的核心手段：
- **有效期设定**: 支持设置 `无限制 (Never expires)`，或自定义礼品卡余额将在售出后 `N 个月内` 自动物理失效。
- **金额卡号生成**: 消费者购买礼品卡后，由系统后台加密库合成唯一的凭证玛，并直接向消费者发送礼品信、说明和激活二维码。
- **与结算对平**: 虚拟卡在结账时可当作现金直接扣减实盘应付总额。

## 2. Metafields 扩展属性定制 (Dati Personalizzati)
Shopify 原生的产品、订单、客户数据表已经定义了核心字段，但企业往往需要存储个性化的特殊数据。AI Commerce OS 的多租户元字段引擎可实现物理数据列扩展：
- **Scope (作用域)**:
  - **Products**: 可用于定义 CE 欧盟安全认证链接 (`custom_ce_label`)、成分大类、洗涤说明 (`fabric_care`) 等。
  - **Orders**: 可定义货代专用舱单标记。
  - **Customers**: 用于存储高净值企业的退税 EORI 税号 (`custom_eori_tax`)。

## 3. 多语言本地化管理 (Lingue)
- **多大区多极体验**: 
  - 支持多语种对齐。例如默认使用英语作为后台基础，在面对法、意、德等本土消费者时，渲染对应国家的翻译内容。
  - 所有翻译均支持在前台翻译引擎处一键切换，AI 翻译智体可以协助店长一键本地化所有 SKU 标题和详情。

## 4. 隐私安全与 Cookie 同意
- 整合了对 GDPR 同意声明 (Cookie Banner) 的状态微调：
  - **🛡 强力阻断 (Strict Opt-In)**: 在访客同意前，不泄露任何 IP/追踪。
  - **柔和提示 (Alert Banner)**: 允许基本的首轮统计。
