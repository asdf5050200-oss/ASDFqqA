# 📍 地点/仓库 (Sedi) - 多仓物理储能与库存调度指南

在多渠道零售链路中，准确物理分拨和控制多储库网点 (Locations / Sedi) 的库存状态，是保证按期完成派单并防范爆仓/断货风险的生命线。

---

## 1. 添加与开通多实体仓储网点
- 系统支持挂接并管理多处物理据点：
  - **零售门店实体据点 (Physical Retail Store)**: 在南欧、北欧大区开设的品牌旗舰店，兼具自提与物理零售体验功用。
  - **骨干保税仓配中心 (Distribution Center / Warehouses)**: 例如我们的巴黎主运营保税仓 (`WH-PARIS`)、意法中转枢纽站 (`WH-ROME`) 等高吞吐枢纽。

## 2. 物理库存分配与调度规则 (Regole di prioritizzazione)
- **就近分单算法**: 买家下单后，智体 Oliver 调度算法会解析买家的送达国家，自动配对最合理的库房出库。
- **库存锁定策略**:
  1. 顾客开启结账表单并锁定价格后，对应 SKU 在指定的 Location 被置为“已预留 (Reserved)”。
  2. 收款成功回调后，Reserved 状态正式转为“物理核销扣减”。
  3. 若支付失败或超时，在废弃结账回收期，预留的库存将自动返还至所属库房实盘。

## 3. 多仓库存实机调对
- 在设置面板中可以开启/禁用指定的 Location 活跃状态。
- 如果某海外仓（如：柏林配送中心 `WH-BERLIN`）由于大区物流拥滞需要临时检修停发，可手动将该 Locations 设为不活跃，所有分单指令将自动分流转至备用库站（如：法国巴黎保税仓）。
